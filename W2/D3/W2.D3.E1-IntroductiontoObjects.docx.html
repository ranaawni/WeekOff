Introduction to Objects

Learning Objectives:

  • Understand what objects are.
  • Understand what key/value pairs are.
  • Accessing properties using dot notation & bracket notation/
  • Add/Removing/Modifying properties in an object.
  • Looping through all properties of an object using (for.. in) loop.
  • Using object.keys() object.values().
