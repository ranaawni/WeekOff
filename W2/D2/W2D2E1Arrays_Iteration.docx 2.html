  1. Write a function called shortestOfMixed that takes an array of mixed elements as a parameter, and returns the shortest string within the given array.

Notes:

  1. If there are ties, it should return the first element to appear in the given array.
  2. Expect the given array to have values other than strings.
  3. If the given array is empty, it should return an empty string.
  4. If the given array contains no strings, it should return an empty string.

shortestOfMixed ([4, "two", 2, "three"]) -> "two"

shortestOfMixed ([4, "too", "two", 2, "three"]) -> "too"

shortestOfMixed ([0 , 1, 4]) -> ""
