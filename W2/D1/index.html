Introduction to Arrays

Learning Objectives:

  • Learn about the first type of Data Structure; “Array”
    ”Array is a single variable that is used to store different elements. It is often used when we want to store a list of elements and access them by a single variable.”
  • Learn how to create an array.
  • Learn how to access elements inside an array and learn the meaning of index in an array.
    “    JavaScript arrays are zero-indexed: the first element of an array is at index 0 , and the last element is at the index equal to the value of the array's length property minus 1”
  • Learn about the array special property .length
  • Learn about array functions (methods).

Ex: .pop(), .push(), .shift(), .unshift().

  1. Write a function called push that takes two parameters, an array and a new element, and returns a new array with the element added to the end.
